# 日志

记录日志

## API 接口

### `this.$rtcApi.logger()`

## 示例代码

```js
this.$rtcApi.logger(content, type, productName)
```

## 参数

| 参数        | 说明              | 类型   | 可选值 | 默认值       |
| ----------- | ----------------- | ------ | ------ | ------------ |
| content     | 日志内容 （必选） | string | —      | —            |
| type        | 日志级别 （可选） | string | —      | info         |
| productName | 产品名称（可选）  | string | —      | sc_classroom |
