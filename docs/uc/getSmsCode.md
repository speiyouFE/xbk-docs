# 获取手机验证码

从集团用户中心获取手机验证码

## API 接口

### `this.$ucApi.getSmsCode(phoneCode, phone)`

## 示例代码

```js
this.$ucApi.getSmsCode(86, '13900138000')
```

## 参数

| 参数      | 说明         | 类型   | 可选值 | 默认值 |
| --------- | ------------ | ------ | ------ | ------ |
| phoneCode | 手机号地区码 | number | —      | 86     |
| phone     | 手机号       | string | —      | —      |
