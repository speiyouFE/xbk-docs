# 改变窗口状态

更当前窗口状态

## API 接口

### `this.$nativeApi.changeWindowDisplayState(state)`

> state 取值：1=>最小化，2=>最大化，3=>全屏，0=>正常窗口

## 示例代码

```js
this.$nativeApi.changeWindowDisplayState(1)
```

## 参数

| 参数  | 说明     | 类型   | 可选值     | 默认值 |
| ----- | -------- | ------ | ---------- | ------ |
| state | 窗口状态 | number | 1，2，3，0 | —      |
