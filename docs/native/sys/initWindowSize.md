# 初始化窗口

初始化窗口大小

## API 接口

### `this.$nativeApi.initWindowSize(width, height)`

## 示例代码

```js
this.$nativeApi.initWindowSize(500, 600)
```

## 参数

| 参数   | 说明     | 类型   | 可选值 | 默认值 |
| ------ | -------- | ------ | ------ | ------ |
| width  | 窗口宽度 | number | —      | —      |
| height | 窗口高度 | number | —      | —      |
