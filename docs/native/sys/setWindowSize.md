# 设置窗口大小

设置窗口尺寸

## API 接口

### `this.$nativeApi.setWindowSize(width, height)`

## 示例代码

```js
this.$nativeApi.setWindowSize(500, 500)
```

## 参数

| 参数   | 说明     | 类型   | 可选值 | 默认值 |
| ------ | -------- | ------ | ------ | ------ |
| width  | 窗口宽度 | number | —      | —      |
| height | 窗口高度 | number | —      | —      |
