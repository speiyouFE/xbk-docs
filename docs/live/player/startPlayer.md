# 开始播放

开始播放指定 widgetID 的视频

## API 接口

### `this.$liveApi.startPlayer(widgetID)`

## 示例代码

```js
this.$liveApi.startPlayer(101).then(() => {
  // ...
})
```

## 参数

| 参数     | 说明          | 类型   | 可选值 | 默认值 |
| -------- | ------------- | ------ | ------ | ------ |
| widgetID | 播放器窗口 id | number | —      | —      |
