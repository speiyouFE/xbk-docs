# 设置播放器缓冲时间

设置播放器缓冲时间

## API 接口

### `this.$liveApi.setPlayerBufferTime(widgetID, milliseconds)`

## 示例代码

```js
this.$liveApi.setPlayerBufferTime(101, 5000)
```

## 参数

| 参数         | 说明           | 类型   | 可选值 | 默认值 |
| ------------ | -------------- | ------ | ------ | ------ |
| widgetID     | 播放器窗口 id  | number | —      | —      |
| milliseconds | 缓冲时间，毫秒 | number | —      | 3000   |
